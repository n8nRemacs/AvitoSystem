# tenant-auth / src / routers

**Назначение:** HTTP-роутеры FastAPI — все публичные и защищённые эндпоинты сервиса аутентификации.

**Статус:** WIP.

---

## Роутеры

- `health.py` — `GET /health` (без авторизации)
- `auth.py` — регистрация, вход (OTP-запрос + верификация), refresh токена, logout
- `profile.py` — просмотр и обновление профиля тенанта
- `api_keys.py` — CRUD API-ключей тенанта
- `sessions.py` — активные сессии пользователя (не Avito-сессии, а auth-сессии)
- `invites.py` — создание и использование инвайт-ссылок
- `billing.py` — подписка, тарифы, история платежей
- `notifications.py` — настройки уведомлений
- `tenant_params.py` — параметры тенанта (настройки, лимиты)
