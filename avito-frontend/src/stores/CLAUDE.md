# avito-frontend / src / stores

**Назначение:** Pinia stores — глобальное состояние SPA.

**Статус:** not_used_in_v1.

---

## Stores

- `auth.js` — access_token, refresh, login/logout actions
- `tenant.js` — данные тенанта, профиль
- `search.js` — результаты поиска, параметры фильтрации
- `messenger.js` — каналы, сообщения, SSE-подписка на real-time события
- `farm.js` — список устройств, статусы токенов
