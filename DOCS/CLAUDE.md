# DOCS

**Назначение:** Вся документация проекта AvitoSystem — API-спецификации, архитектура, гайды, ТЗ.

**Статус:** актуальна. Основные документы соответствуют состоянию на апрель 2026.

---

## Навигация

### Главные документы

| Файл | О чём |
|---|---|
| `TZ_Avito_Monitor_V1.md` | **Главный документ V1.** ТЗ системы мониторинга: стек, архитектура, функциональные требования, модули (Search Profiles, Price Intelligence, MCP, Token Farm). Читать первым |
| `AVITO-API.md` | Полная спецификация реверс-инженерного API Avito: заголовки, токены, Messenger HTTP/WS JSON-RPC, поиск, QRATOR, извлечение токенов. Актуально на февраль 2026, версия приложения 215.1 |
| `X-API.md` | Документация собственного avito-xapi: все эндпоинты, авторизация, модели данных |
| `ARCHITECTURE.md` | Инфраструктура: Proxmox homelab, VPS relay, SSH-туннели, Cloudflare DNS, сетевая схема |

### Гайды

| Файл | О чём |
|---|---|
| `AVITO-FINGERPRINT.md` | Устройство fingerprint Avito, как подменять через Frida |
| `REVERSE-GUIDE.md` | Процесс реверс-инжиниринга Avito Android: Frida, mitmproxy, SSL bypass |
| `DEPLOY_PLAN.md` | План деплоя на homelab |
| `EXTENDING.md` | Как расширять систему (новые провайдеры, модули) |
| `MOCKS.md` | Описание мок-данных для разработки без реального Avito |

### Отчёты / специальные

| Файл | О чём |
|---|---|
| `REPORT.md` | Отчёт об исследовании (прогресс реверс-инжиниринга) |
| `SYSTEM.md` | Описание системы (ранняя версия архитектуры) |

| `token_farm_system.md` | Описание token farm: устройства, агент, ротация токенов |

### avito_api_snapshots/

JSON-снимки реальных ответов Avito API: дерево категорий, поля фильтрации для телефонов, наушников, планшетов, умных часов. Используются как справочник при построении поисковых запросов. Есть `README.md` внутри.
